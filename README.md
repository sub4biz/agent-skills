# Agent Skills

My collection of skills for AI coding agents. Skills are packaged instructions and scripts that extend agent capabilities.

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

- **[idea-refine](skills/idea-refine)** - Refine ideas through structured divergent and convergent thinking.

## Installation

### Claude Code

```bash
cp -r skills/idea-refine ~/.claude/skills/
```

### claude.ai

Add the skill to project knowledge or paste `skills/idea-refine/SKILL.md` contents into the conversation.
